# PHP Student Notes Summary

A small plain PHP practice project built to strengthen core PHP fundamentals such as variables, arrays, associative arrays, loops, functions, and browser output.

## Goal

Build a simple multi-page PHP project that:
- shows all student notes
- filters open notes
- shows a status summary

## Purpose

This project is designed to practice:
- variables
- arrays
- associative arrays
- loops
- functions
- basic conditional logic
- PHP syntax differences from JavaScript
- rendering PHP output in the browser

## Planned Files

- `data.php`
- `functions.php`
- `index.php`
- `open-notes.php`
- `status-summary.php`
